# Change Log

All notable changes to the "prettier-eslint" extension will be documented in this file.

## [1.0.0] - 2025-11-08

### Added
- Initial release
- Format document command
- Format on save functionality
- Support for JavaScript, TypeScript, JSX, TSX, JSON, and JSONC
- Configuration options for enabling/disabling formatter
- Integration with project's Prettier and ESLint configurations
